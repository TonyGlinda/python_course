# 程序结构 顺序、分支、循环
- 分支语句
    - if分支语句
        - 基本语法：
            - if 条件表达式：
                - 语句1（条件成立时要做的事情）
                - 语句2 （条件成立时要做的事情）
                  ......
        - 条件表达式就是计算结果必须为布尔值的表达式
        - 表达式后面的冒号不能少
        - 条件表达式结果为True执行if后面的缩进的语句块
        
    - 双向分支 if......else......
        - if 条件表达式：
            - 语句1
            - 语句2
            - ......
        -   else：
             - 语句1 （条件不成立是要做的事）
             - 语句2
             - ......
             
    - 多向分支 
        - if 条件表达式：
            - 语句1
        - elif 条件表达式：
            - 语句2
        - elif 条件表达式：
            - 语句3        
            
    - elif可以有很多个
    - else可选
    - 多路只会选择一个执行。     
    
- if语句的嵌套
    - 在之前的条件成立时，还需要进行条件判断时使用。
