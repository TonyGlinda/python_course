# 算数运算符
- 运算符
    - 加法        使用 + 
    - 减法        使用 -
    - 乘法        使用 * 
    - 除法        使用 / 
    - 整除        使用 //       
    - 取模（余数）使用 % 
    - 幂运算      使用 ** 
    
- 运算优先级问题
    - 和数学中的运算优先级一致。
        - 先乘除后加减
        - 同级运算，从左往右
        - 可以使用括号调整优先级。
    - 先：幂运算**
    -    正负号 +a -a
    -   算数操作：先  *  /  //  % 再 + - 
    -   比较操作符： <  <=  >  >=  ==  !=
    - 后：逻辑运算： not  and  or  
    
- 算数运算的表达式
    - a = a + 3    简写  a + = 3
    - a = a - 3    简写  a - = 3
